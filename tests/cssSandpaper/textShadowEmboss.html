An example of embossed text-shadows with a rotate transform thrown in. Note that in order for the transform to work, we must set the <body> tag to cssSandpaper-noMSTransform in order to force cssSandpaper to use the Matrix Visual Filter instead of IE9's native CSS -moz-transform CSS property, since this breaks the text-shadow.
